A Laravel-based SEO crawler focused on internal linking analysis, built with DTOs, queue-safe crawling, and rate limiting.

This project is designed to be the core engine for SEO tools such as:
	•	Internal linking opportunity detection
	•	Orphan page discovery
	•	Link depth & authority analysis
	•	Multilingual site audits

Core principles
	•	Separation of concerns
	•	DTO-based crawler output
	•	Queue-first design
	•	No external APIs
	•	Deterministic & testable
	•	Scales horizontally
